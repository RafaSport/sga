# Sistema de Gestão de Abrigos

Rede de Acolhimento — Prefeitura do Recife

## Etapa 2 — Levantamento de Requisitos

### 1. Introdução

Requisitos funcionais e não-funcionais do SGA.

### 2. Requisitos Funcionais

- **RF01–RF12:** Prontuário, documentos, transferência e desligamento
- **RF13–RF16:** Medicações e prescrições
- **RF17–RF19:** Agendamentos
- **RF20–RF23:** Acesso, perfis e layout
- **RF24–RF25:** Painel gerencial
- **RF26–RF27:** Relatórios estatísticos

### 3. Requisitos Não-Funcionais

- **RNF01:** Usabilidade simples
- **RNF02:** Responsividade
- **RNF03:** Desempenho em conexões básicas
- **RNF04:** Segurança e LGPD
- **RNF05:** Rastreabilidade
- **RNF06:** Disponibilidade

### 4. Resumo dos Requisitos

| Código      | Descrição               | Prioridade |

| RF01–RF12   | Prontuário e documentos | Alta       |
| RF13–RF16   | Medicações              | Alta       |
| RF17–RF19   | Agendamentos            | Alta       |
| RF20–RF23   | Acesso e layout         | Alta       |
| RF24–RF25   | Painel gerencial        | Alta       |
| RF26–RF27   | Relatórios estatísticos | Média      |
| RNF01–RNF06 | Não-funcionais          | Alta       |
